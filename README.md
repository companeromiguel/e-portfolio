## Getting Started

```bash
cd parallax-portfolio
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view your portfolio.

## Customize

Edit `app/page.tsx` to update your content:
- Hero section with your name
- About & Skills sections
- Projects showcase
- Contact section

The gradient background automatically creates parallax depth as you scroll.
